Pokémunk

An immersive and engaging JavaScript game inspired by the popular Pokémon franchise. This game utilizes HTML canvas to provide a top-down view of detailed maps and enables players to explore a rich and diverse game world. With player boundary collision detection and exciting battle sequences, this game brings the thrilling Pokémon experience to life.

Notes App

This is a simple notes app made using React that lets you create, save and delete notes. The app uses local storage to store your notes so that you don't lose them even if you close the app or refresh the page. The app has a user-friendly interface with a toggle button that lets you switch between light and dark mode. This ensures that you can use the app comfortably in any lighting condition. With this app, you can create multiple notes, edit their content, and delete them as you see fit. The app also comes with a search feature that lets you quickly find notes by their title or content. This app is perfect for anyone who needs a simple, lightweight notes app that they can use to jot down ideas, make to-do lists, or keep track of important information.

Text Encryption

This project is a simple text encryption application built using HTML, CSS, and Javascript. It utilizes a basic substitution cipher to encrypt and decrypt text, replacing specific letters with unique sequences. The app also boasts user-friendly features like converting uppercase letters to lowercase before encryption, displaying results in a dedicated output field, and offering buttons to copy encrypted/decrypted text or clear the input field. This project demonstrates skills in UI design, code organization, and implementing the encryption/decryption logic, making it a valuable learning experience for beginners in web development.
